# E-Commerce Data Analysis using Excel

##  Project Overview
This project focuses on cleaning and analyzing an e-commerce dataset using Microsoft Excel.  
The objective is to work with messy data, perform structured analysis, and answer business-related questions.

---

##  Tools Used
- Microsoft Excel  
- Excel formulas  
- Pivot Tables  

---

##  Analysis Process
1. Cleaned raw data by handling missing values, duplicates, and inconsistent entries  
2. Standardized columns and prepared structured tables for analysis  
3. Created pivot tables to analyze sales, orders, customer segments, and order status  
4. Performed trend and comparative analysis across months, categories, and channels  
5. Derived clear insights to support business and operational decisions  

---

##  Key Analysis Areas
- Monthly sales and order trends  
- Customer segmentation by gender and age group  
- Top-performing states and cities  
- Channel-wise order distribution  
- Order status (delivered, cancelled, returned)
